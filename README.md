# 弹弹堂 65° 力度计算器

一个离线 Android 力度计算器。输入水平距离、风力、风向、高度差和当前角度，即可得到推荐力度及建议范围。

## 下载 APK

[点击下载最新版 APK](https://github.com/amadeus670/dandanxiuqiu/releases/download/latest/dandanxiuqiu-debug.apk)

首次安装时，Android 可能提示允许浏览器或文件管理器“安装未知应用”。

## 计算模型

- 基础力度来自游戏说明中的 65°、零风、同高度数据表。
- 非整数距离使用相邻数据线性插值。
- 顺风减力度、逆风加力度。
- 目标高于自己时增加力度，低于自己时减少力度。
- 角度以 65° 为基准修正。
- 风力、高度和角度修正系数可在应用内调整并自动保存。

默认公式：

```text
推荐力度 = 基础力度
         + 风向符号 × 风力 × 0.55
         + 高度差 × 1.20
         + (当前角度 - 65) × 0.65
```

不同服务器和武器的弹道可能不同，默认系数仅供参考，建议根据实际命中数据校准。

## 自动构建

每次推送到 `main` 后，GitHub Actions 会：

1. 使用 JDK 17、Gradle 9.5 和 Android Gradle Plugin 9.3；
2. 运行单元测试；
3. 构建 debug APK；
4. 将 APK 上传为 Actions 构建产物；
5. 更新 `latest` Release 中的直接下载文件。
